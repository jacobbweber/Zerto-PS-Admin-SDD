# Interface Contracts: ZertoZVM.APIWrapper

**Date**: 2026-04-05
**Feature**: 001-zertoapiwrapper-module

## Primary Proxy Contract 
`Invoke-ZertoRequest`

This private function implements the singular outbound path for all REST requests bound for the ZVM.

**Input**
```powershell
[CmdletBinding()]
param(
    [Parameter(Mandatory=$true)]
    [ValidateSet('GET', 'POST', 'PUT', 'DELETE')]
    [string]$Method,

    [Parameter(Mandatory=$true)]
    [string]$UriPath,

    [Parameter()]
    [object]$Body,

    [Parameter()]
    [hashtable]$QueryParameters
)
```

**Responsibilities**
- Appends `UriPath` directly to `$script:ZertoSession.BaseUri` using `[System.UriBuilder]`.
- Compiles `QueryParameters` natively using `[System.Web.HttpUtility]::ParseQueryString`.
- Ignores executing Payload conversions when `$Body` passes as `$null`.
- Executes HTTP via `Invoke-RestMethod` appending `-Headers $script:ZertoSession.Headers` and checking `-SkipCertificateCheck` where authorized.

**Outputs**
- Always returns raw Deserialized `[PSCustomObject]` arrays directly generated by `Invoke-RestMethod` JSON conversion or void on 204.
- Throws strongly typed terminating PowerShell Exceptions explicitly outlining Method, Full Built URI, and captured `[int]` HTTP response codes upon failures. 

## Command Standard Contract 

All externally facing functions enforcing API reads must follow:

### Definition Standard
```powershell
function Get-ZertoNoun {
    [CmdletBinding()]
    param(
        [Parameter(Mandatory=$false)] [string]$CustomFilter
    )

    Assert-ZertoSession

    $QueryParams = @{}
    if ($null -ne $CustomFilter) { $QueryParams['filter'] = $CustomFilter }

    return Invoke-ZertoRequest -Method 'GET' -UriPath '/v1/noun' -QueryParameters $QueryParams
}
```

### Async Modification Standard 
```powershell
function Start-ZertoAction {
    [CmdletBinding(SupportsShouldProcess)]
    param(
        [Parameter(Mandatory=$true)] [string]$Id
    )
    
    Assert-ZertoSession

    if ($PSCmdlet.ShouldProcess("Resource: $Id", "Start-Action")) {
        return Invoke-ZertoRequest -Method 'POST' -UriPath "/v1/noun/$Id/action"
    }
}
```
